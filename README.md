# Workout Assistant

## Installation

```bash
git clone git@github.com:wandac/workout-assistant.git
cd WorkoutAssistant
npm install
```

## To run the project

```bash
expo start
```

### Building standalone Android app
1. Configure app.json - increment app version
2. ```expo build:android```
